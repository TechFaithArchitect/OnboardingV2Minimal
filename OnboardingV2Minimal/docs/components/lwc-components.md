# LWC Components

Components present in this repo (`force-app/main/default/lwc/`).

## Dashboard & Work Queue
- `onboardingWorkQueue` - Queue list for active onboarding items
- `onboardingVendorProgramGrid` - Vendor program table
- `onboardingKpiRow` - KPI tiles row
- `onboardingInsights` - Insights panel
- `onboardingRecentActivity` - Activity stream widget

## Onboarding Modals & Entry
- `onboardingDealerOnboardingModal` - Modal for "Start Dealer Onboarding"

## Onboarding Records
- `onboardingOrderStatusViewer` - Order/training status summary
- `onboardingECC` - External contact credential detail view

## Rules / modals (Admin)
- `onboardingRuleModal` - Modal wrapper for rule edits

## Program Dates & Related
- `programDatesScreenAction` - Program dates screen action
- `programDatesRelatedList` - Program dates related list
- `programDatesQuickAction` - Program dates quick action
- `programDatesRelatedListWrapper` - Wrapper for program dates

## Selectors & Editors
- `vendorSelector` - Vendor selector
- `vendorProgramSelector` - Vendor program selector
- `recordCollectionEditor` - Record collection editor
- `objectRelatedList` - Reusable related list
- `objectRelatedListDatatable` - Datatable for related lists
- `relatedObjectActionModal` - Modal for related object actions

## Path & Override
- `onboardingPathSelector` - Path selector
- `overrideAuditPanel` - Override audit log UI

## Messaging & Admin
- `twilioSettings` - Twilio configuration UI

## Utilities
- `utils` - Shared helpers used by multiple LWCs
- `reactStylePhoneInput` - Phone input component
