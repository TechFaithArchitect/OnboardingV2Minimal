# Vendor Onboarding Wizard API (Deprecated)

The vendor onboarding wizard and its APIs were removed in the MVP v2 scope. This document is retained only to indicate the removal.

Use the current onboarding entry points instead:
- `accountProgramOnboardingModal` (LWC)
- `OnboardingRequirementsPanelController` (Apex)
- `OnboardingStatusRulesEngineController` (Apex)
